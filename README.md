## Overview

Verdant Plant is an open AI plant care network. IoT sensors connect to Claude AI which autonomously monitors, analyzes, and acts on your plant's environment — watering, lighting, ventilation — without any human intervention.

Trading fees from $VRDT on Solana fund the AI infrastructure. No subscriptions. Fully transparent on-chain treasury.

## Deploy

```bash
# Clone
git clone https://github.com/verdantplant/website
cd website

# Open locally
open index.html

# Deploy — just upload all .html files to your hosting
```

No build process needed. Upload the 4 HTML files directly to any static host.
